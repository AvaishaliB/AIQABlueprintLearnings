# Task Plan

## Blueprint

### Tech Stack
- **Frontend**: React with TypeScript.
- **Backend**: Node.js with TypeScript (Express).

### Features
- **UI**: Main chat interface with History sidebar, test case output, and input area for Jira requirements. Settings panel for configuring LLM infrastructure.
- **LLM Integrations**: Ollama, LM Studio, Groq, OpenAI, Claude, Gemini. Includes API key/endpoint management and "Test Connection" functionality.
- **Output Generator**: Yields Functional and Non-Functional test cases (API and Web App) formatted specifically for Jira.

## Phases
- **Phase 1**: Discovery and Initialization (Completed)
- **Phase 2**: Scaffold Project (React TS frontend, Node TS backend)
- **Phase 3**: Build UI Components (Main Chat, Settings Modal)
- **Phase 4**: Implement LLM Integrations & Backend Routes
- **Phase 5**: Prompt Engineering for Jira Test Case Formats
